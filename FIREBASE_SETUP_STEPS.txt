WE CAN STEP — Firebase Setup Steps

تم تحديث النسخة على إيميل الأدمن:
support@wecan4step.com

1) Authentication
- Build > Authentication > Sign-in method
- فعّل Email/Password فقط.
- من تبويب Users اضغط Add user
- Email: support@wecan4step.com
- Password: كلمة مرور قوية تختارها أنت ولا ترسلها لأحد.

2) Firestore Database
- أنشئ Firestore بالوضع Production mode.
- الموقع المقترح: eur3 (Europe).

3) Firestore Rules
الصق القواعد التالية في:
Firestore Database > Rules > Edit rules > Publish

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {

    function signedIn() {
      return request.auth != null;
    }

    function isAdmin() {
      return signedIn() && request.auth.token.email == "support@wecan4step.com";
    }

    match /subscriptionRequests/{requestId} {
      allow create: if signedIn()
        && request.resource.data.keys().hasOnly(['uid', 'name', 'phone', 'email', 'status', 'source', 'createdAt', 'createdAtMs'])
        && request.resource.data.uid == request.auth.uid
        && request.resource.data.email == request.auth.token.email
        && request.resource.data.status == 'pending'
        && request.resource.data.source == 'website';

      allow read, update, delete: if isAdmin();
    }

    match /students/{studentEmail} {
      allow read: if isAdmin()
        || (signedIn() && resource.data.email == request.auth.token.email);

      allow create, update, delete: if isAdmin();
    }

    match /phoneLookup/{phone} {
      allow get: if true;
      allow list: if false;
      allow create, update, delete: if isAdmin();
    }
  }
}

ماذا تم ربطه في النسخة:
- صفحة طلب الاشتراك تحفظ الطلب داخل Firestore.
- يتم إنشاء حساب الطالب في Firebase Authentication عند إرسال الطلب.
- لوحة الإدارة تقرأ الطلبات من Firestore.
- زر قبول يفعّل الطالب داخل students ويربط جواله في phoneLookup.
- زر رفض يغيّر حالة الطلب إلى rejected.
- زر إضافة طالب يدويًا ينشئ حسابًا للطالب ويفعّله مباشرة.
- تسجيل الدخول يدعم الإيميل أو رقم الجوال للطلاب المقبولين.
- دخول لوحة الإدارة يكون عبر support@wecan4step.com وكلمة مرور Firebase التي وضعتها أنت.
