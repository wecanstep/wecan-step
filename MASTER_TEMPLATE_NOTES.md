# WE CAN Reading Study Master Template

تم اعتماد هذه النسخة كقالب رسمي نهائي لنماذج Reading القادمة.

## المرجع
Reading Model 49

## العناصر المعتمدة
- Test Mode رسمي.
- Study Mode كامل.
- خطة مذاكرة النموذج.
- صفحة إنجاز الطالب.
- وصف الفرق بين:
  - اختبر النموذج = محاكاة اختبار حقيقية
  - ذاكر النموذج = شرح وتدريب وفهم الإجابات وترجمتها
- تكبير اسم النموذج في كروت Reading.
- إظهار اسم النموذج في صفحة النتيجة بصيغة: 📖 Reading Model 49.

## ترتيب شرح Study Mode
1. معنى الإجابة.
2. لماذا هذه الإجابة؟
3. الدليل من القطعة مع رقم الفقرة.
4. ترجمة الدليل مباشرة تحت الدليل.
5. العلاقة بين الإجابة والدليل.
6. لماذا الخيارات الأخرى خطأ؟ في آخر الشرح.

## ملاحظة
هذه النسخة محفوظة كمرجع، وأي نموذج Reading جديد يتم بناؤه على نفس الأسلوب بدون تغيير التصميم أو الهيكل.
